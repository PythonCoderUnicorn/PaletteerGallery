# Paletteer Gallery
my visual color palette gallery for paletteer R pkg

```
ggplot(
  mtcars,
  aes(x=mpg,
      y= hp,
      color= cyl)
)+
  geom_point(size= 3)+
  scale_color_paletteer_c("ggthemes::Blue-Green Sequential")+
  ggdark::dark_mode()+
  labs(title = " ggthemes::Blue-Green Sequential ")
```
